# 01-andmeinseneeria-alused

Selles kaustas on teema "Andmeinseneeria alused" materjalid.

- [Baastase](./baastase/README.md) keskendub PostgreSQL-iga ühenduse loomisele, esimese CSV-faili laadimisele ja töövoo põhiloogikale.
- [Edasijõudnud](./edasijoudnud/README.md) on selle teema kiirema tempo ja suurema iseseisvusega rada.

Baastaseme esimese praktikumi tugifailid asuvad kaustas [baastase](./baastase/):

- [compose.yml](./baastase/compose.yml)
- [.env.example](./baastase/.env.example)
- [data](./baastase/data)
- [scripts](./baastase/scripts)
